# Electronics and Computer Science Club (ECSC) - University of Kelaniya

Welcome to the official repository for the Electronics and Computer Science Club website of the University of Kelaniya. This platform serves as a hub for our community, showcasing our events, achievements, and the talented team behind our initiatives.

![ECSC Logo](public/assets/logo/ecsc_logo_header.png)

## üöÄ Overview

The ECSC website is designed to connect students, alumni, and tech enthusiasts with the latest updates from our club. It features a modern, responsive design with interactive elements to provide an engaging user experience.

## ‚ú® Features

- **Interactive Hero Section**: Engaging introduction with animated elements.
- **Core Events Showcase**: Dynamic section highlighting our major annual events (Robot Battles, Medusa CTF, etc.) with interactive hover effects.
- **Achievements Gallery**: A visual celebration of our club's milestones and student victories.
- **Meet the Team**: A dedicated page introducing the Executive Board and committee members.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Smooth Animations**: Custom wavy underlines and blob animations for a unique aesthetic.

## üì¶ Installation & Setup

Follow these steps to set up the project locally:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/ecs-uok/ecs-uok-website.git
    cd ecs-uok-website
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.
